## Reporting Issues

If you find a bug or something does not work, please create a GitHub Issue.

Please include as much information as possible:

- **Date and time** when the issue happened
- **Server IP** used
- **Test mode:** no-auth, LAN, or auth
- **Player name/persona**
- **What you were trying to do**
- **What actually happened**
- **What you expected to happen**
- **Whether you were** in the lobby, a room, or a race
- **Screenshots, logs, or videos,** if available

### Example Issue Format

```
Date/time: 2026-05-01 21:35
Server IP: 161.35.110.36
Test mode: no-auth
Player name: TestPlayer

What I did:
I joined the lobby and tried to create a room.

What happened:
The room appeared for me, but my friend could not see it.

Expected result:
The room should appear in the room list for other players.

Extra info:
The game did not crash.
```
